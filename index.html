    reach-api

    A Ruby wrapper for the Halo Reach stats API.

    installation

    [sudo] gem install reach-api

    usage

    You'll need a Bungie-provided API key:

    require "reach-api"
reach = Reach.new("your_key")

    For information on API methods, please refer to the wiki.
